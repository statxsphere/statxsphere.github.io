Product: NBA Draft Evaluation V2

Code | Website

Skills used: Web Scraping, Object Oriented Programming, Averaged Regression Models, Clustering, PCA, Cosine Similarity, Deployment Via Streamlit, Dashboarding.

Data: Scraped from tankathon.com and basketball-reference.com using bs4 and nba.com using APIs.

Improvements:

  • New labels were used as the regression output: Peak Impact Rating and Immediate Impact Rating.
  • Peak Impact Rating: highest 3-season rolling average VORP, extrapolated to a (-10,10) range.
  • Immediate Impact Rating: average first 2 season VORP, extrapolated to a (-10,10) range.
  • Clustering and similarity was used to find player comps for their ceiling and floor.
  • Dashboard of the above info was deployed using Streamlit.
